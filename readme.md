### 项目描述
前端: vue+vue-router+vuex+socket.io+typescript

后端: nestjs+nestjs/websocket+typescript

### 功能介绍
- 更改用户名/头像上传
- 群聊/私聊
- 新增群/加入群聊/模糊搜索群
- 添加好友/模糊搜索好友
- 表情包
- 消息分页

### 环境配置
- 数据库mysql 以及 chat数据库
- node v10.16.3

### 运行项目
前端项目
```js
cd genal-chat-client 
npm run serve
```
后端项目
```js
cd genal-chat-server
npm run start:dev
```
