### 项目描述
前端: vue+vue-router+vuex+socket.io+typescript

后端: nestjs+nestjs/websocket+typescript

### 环境配置
mysql 以及 chat数据库

### 功能介绍
- 新增用户
- 新增群
- 群聊

### 运行
前端项目
```js
cd genal-chat-client 
npm run serve
```
后端项目
```js
cd genal-chat-server
npm run start:dev
```
